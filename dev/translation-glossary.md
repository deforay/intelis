# InteLIS Translation Glossary

This glossary provides standardized translations for medical and laboratory terms used in the InteLIS (Integrated Laboratory Information & Sample Tracking System).

## General Guidelines

- **Keep acronyms unchanged**: DBS, PCR, TAT, ART, ARV, VL, EID, CD4, HBV, HCV
- **Use context-appropriate translations**: Medical terms must reflect clinical meaning, not literal translation
- **Maintain consistency**: Use the same translation for the same term throughout

## Translation Quality Checklist

When translating, verify:

1. **Accents**: Ensure proper French accents (é, è, ê, à, â, ù, û, ç, ï, î, ô)
2. **Gender agreement**: Match adjective gender with noun
3. **No mixed languages**: Avoid "Résultat PDF Settings" - translate fully
4. **Medical accuracy**: "Suppressed" in viral load context means undetectable, not deleted
5. **Consistency**: Same English term = same French translation throughout
6. **Natural phrasing**: Translate meaning, not word-by-word
